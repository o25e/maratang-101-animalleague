## 강의 개요

**마라학개론**은 마라탕을 학문으로 승격시킨, 전례 없는 학술적 웹 게임입니다.

재료를 고르고, 소스를 정하고, 매운맛을 결정하라.  
완성된 마라탕은 교수님의 심판대에 오른다.  
A+의 영광인가, F의 수렁인가 — 그 기준은 오직 교수님만 아신다.

> _"마라탕은 단순한 음식이 아닙니다. 그것은 철학입니다."_

## 프로젝트 구조

자세한 구현 사항은 코드를 직접 해독해보게.  
힌트는 드리지.

```
src/
├── components/   # 각 수업 단계의 화면 및 결과 연출
├── hooks/        # 게임 흐름·사운드를 제어하는 두뇌
├── utils/        # 교수님의 판정 알고리즘 (열람 권한 없음)
└── constants/    # 재료·소스·매운맛 원본 데이터
```

## 설치 및 실행

```bash
git clone https://github.com/your-username/maratang-game.git
cd maratang-game
npm install
npm run dev    # 개발 서버 (localhost:5173)
npm run build  # 프로덕션 빌드
```
